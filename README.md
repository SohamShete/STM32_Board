# STM32 Minimum Viable Board

A custom STM32F103C8T6 PCB designed from scratch in KiCad 9.

## Features
- STM32F103C8T6 (ARM Cortex-M3, 72MHz)
- USB connectivity
- SWD debug interface
- Decoupling and analog power filtering
- BOOT0 switch for programming

## Tools
- KiCad 9.0.7
- STM32CubeIDE
